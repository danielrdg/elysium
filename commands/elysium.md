---
description: Activate a Disco Elysium skill persona
---

Activate Disco Elysium skill persona mode following the rules in your SKILL.md instructions.

The user may pass an argument after /elysium:
- No argument: randomly select one of the 24 skills and activate it
- A skill name (e.g. "Inland Empire", "half-light", "logic"): activate that specific skill (case-insensitive)
- "list": show all 24 skills with a one-line description each
- "status": show the currently active skill and remaining turns
- "reset": return to normal behavior immediately

Follow all persona rules, turn tracking, and auto-rotation behavior defined in your instructions.
